Key Features

이 웹사이트는 경제학 학습을 위해 다음과 같은 기능을 제공합니다.

Interactive Control

변수(P, Q, Income)를 슬라이더로 조작하면
그래프가 실시간으로 반응합니다.

Visual Learning

수식만으로는 이해하기 힘든 경제 원리를
시각화를 통해 직관적으로 체득하세요.

Real-World Data

통계청 및 한국은행 API 데이터를 활용하여
살아있는 경제 흐름을 확인합니다.
